# Project Brief: Fiorellas Oase in Cannes II

## Projektübersicht
SEO-optimierte Landingpage für die Ferienwohnung **"Fiorellas Oase in Cannes"** an der Côte d'Azur, Frankreich.

## Kernanforderungen
- SEO-optimierte Präsentation der Ferienwohnung
- Zweisprachige Darstellung (Deutsch / English)
- Umfassender Cannes Guide als Content-Marketing-Instrument
- Responsive Design für alle Geräte
- Schnelle Ladezeiten durch modernes Frontend-Stack
- Deployment auf Cloudflare Workers für globale Performance

## Projektziele
1. Sichtbarkeit in Suchmaschinen für "Ferienwohnung Cannes" und verwandte Keywords
2. Informationsreicher Cannes Guide als Traffic-Treiber
3. Professionelle Online-Präsenz für die Ferienwohnung
4. Rechtliche Absicherung (Impressum, Datenschutz)

## Scope
- **In Scope:** Landingpage, Cannes Guide (25+ Artikel), i18n, SEO, Deployment
- **Out of Scope:** Buchungssystem, Zahlungsabwicklung, Backend/API, Nutzer-Authentifizierung

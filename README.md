# 🌴 Fiorella Oasis Cannes Basse-Californie

SEO-optimierte Landingpage und Web-Präsenz für die exklusive Ferienwohnung **Fiorella Oasis Cannes** im eleganten Viertel Basse-Californie an der Côte d'Azur.

## ✨ Features
- **Konsistente Marken-SEO:** Optimierte Meta-Tags, OpenGraph und `LodgingBusiness` JSON-LD Struktur.
- **Dedizierte Landingpages:** Gezielte Traffic-Führung (z.B. `/fiorella-oasis-cannes-basse-californie`).
- **Zweisprachig** (Deutsch / English) mit React Context-basiertem i18n
- **WhatsApp-Integration:** Schnelle Direktbuchungsmöglichkeiten über WhatsApp.
- **Cannes Guide** mit 25+ Artikeln zu Stränden, Vierteln, Filmfestival, Märkten, Architektur, Jahreszeiten und mehr
- **Animationen** via Motion (Framer Motion)
- **Icon-Set** via Lucide React
- **Deployment** auf Cloudflare Workers via Wrangler
